# Portfolio

## Description
This is simple Portfolio site designed to showcase my background, experience and projects. It is built using HTML and CSS.

## Features
About Me: Includes a brief introduction about myself.
Project Showcase: Provides a section to showcase my projects with Github repository and links.
Contact Section: Allows visitors to contact me directly.

## Structure
index.html: Main HTML file containing the structure of the portfolio site.
styles.css: CSS file containing styles for the site.
img/: Directory containing images used in the portfolio site.
